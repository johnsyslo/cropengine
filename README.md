## Overview
CropEngine is a lightweight Fabric mod designed to automate farming on Hypixel Skyblock. With a focus on efficiency, it removes the repetitive action of farming while maintaining high performance and minimal setup.

## Features
- S-Shaped melon and pumpkins.
- S-Shaped and vertical Carrots, etc.
- Automatic row switching with randomized delays to mimic human-like behavior.
- Easy-to-configure menu with Cloth Config.

## Commands
- `/cropengine`
- `/look <yaw> <pitch>`
- `/yeet`

## Usage
- Press `K` to start the mod.
- Press `Right Shift` to open the config.
